# Amari slideshow

Four-slide Next.js 16 pitch deck for Amari, a local voice interface that can browse the web and turn fresh context into a clear answer.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

Use the arrow buttons, `←` / `→`, Space, Home, End, or the autoplay control. The deck uses a sky-blue/ocean palette, a responsive mobile menu, reduced-motion support, and animated transitions between the four scenes.
